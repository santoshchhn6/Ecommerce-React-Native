## Prerequisite
### Firebase
1. Create Firebase Account

2. Enable Gmail and Password Authentication.

3. Enable Firestore Database.

4. Enable Firebase Storage.

### Add Product
Goto root directory of project.

$ cd admin

$ npm install

$ npm run dev

goto http://localhost:5173/

Add Product
